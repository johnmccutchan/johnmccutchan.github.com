About

  • American born Canadian. Grew up in Kitchener-Waterloo, Ontario, Canada
  • Received an M.Sc. in Computer Science from McMaster University
  • Lives in Berkeley, California

Major Works

  • Built a live development environment for the Dart VM with state preserving hot reloading of source code, provides sub-second iteration times for Dart and Flutter (2016)
  • Lead of Dart VM's performance analysis tool suite: Observatory (2014)
  • Defined SIMD computation model for dynamically typed programming languages (2013)
  • Created a SIMD programming model for the Dart programming language and implemented it in the Dart Virtual Machine, providing 4x speed up to real world applications (2012)
  • Creator of Move.Me which allowed independent developers to make software that uses PlayStion Move controllers (2011)
  • Worked on Bullet physics engine (2008)
  • Created inotify, an efficient file system notification subsystem for the Linux kernel (2005)
